## Simple Chess game 

Simple chess game that can be played in terminal, of course with two player and in a single machine.

```
       _a_ _b_ _c_ _d_ _e_ _f_ _g_ _h_ 
0 -->  Br1 Bn1 Bb1 Bq1 Bk1 Bb2 Bn2 Br2
1 -->  Bp1 Bp2 Bp3 Bp4 Bp5 Bp6 Bp7 Bp8
2 -->  ___ ___ ___ ___ ___ ___ ___ ___
3 -->  ___ ___ ___ ___ ___ ___ ___ ___
4 -->  ___ ___ ___ ___ ___ ___ ___ ___
5 -->  ___ ___ ___ ___ ___ ___ ___ ___
6 -->  Wp1 Wp2 Wp3 Wp4 Wp5 Wp6 Wp7 Wp8
7 -->  Wr1 Wn1 Wb1 Wq1 Wk1 Wb2 Wn2 Wr2
```

### Running
No need to install any modules.

```commandline
python3 chess.py
```
